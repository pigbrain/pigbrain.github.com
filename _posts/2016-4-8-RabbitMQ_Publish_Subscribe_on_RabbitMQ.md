# Publish/Subscribe (using the Java Client)  
* 여러 Consumer에게 메세지를 전달하는 방법을 소개한다  
	* 이러한 패턴을 Publish/Subscribe라고 한다  
	* 로깅시스템을 예로 들어서 소개 할 것이다  
	* Publishing된 로그 메세지는 모든 Receiver에게 전달된다
