### Clone
```
$ git clone https://github.com/pigbrain/pigeye.git [target directory]
```

### Commit
```
git commit -a "commit message" // all
git commit file -m "commit message" // specific file
```

### View Staged File
```
git status -v -v
```

### Reset
```
git reset --hard HEAD
```
